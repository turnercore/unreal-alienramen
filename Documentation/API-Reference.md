# C++ API Reference (Doxygen)

The full native API reference is generated by Doxygen from `Source/` during the MkDocs build.

- When building locally, run `doxygen Doxyfile` then `mkdocs serve`.
- On CI (`docs` workflow), Doxygen runs automatically and output is placed under `site/doxygen/`.

Once generated, browse the HTML at [doxygen/html/index.html](/doxygen/html/index.html).
