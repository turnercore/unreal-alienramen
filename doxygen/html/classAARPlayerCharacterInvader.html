Detailed Description

Base ship character for Alien Ramen.

  • ASC is owned by PlayerState, Avatar is this pawn.
  • On server possession: clears loadout, grants common ability set, then resolves ship/hat rows from LoadoutTags and applies baseline. Secondary lane is legacy/deprecated and optional.
  • Exposes generic tag-based activation/cancel API for PlayerController / Blueprint.
